# Resume Buddy – AI Assisted Resume Builder

## 📌 Project Overview

Resume Buddy is a web-based application that helps users create professional resumes easily using predefined chatbot-style commands. Users can enter their personal, educational, and professional details through an interactive interface, and the system automatically generates a structured resume.

The platform also allows users to **download their resume in PDF format** and includes an **ATS (Applicant Tracking System) score checker** that analyzes uploaded resumes to determine how well they are optimized for modern recruitment systems.

---

## ✨ Features

* 🤖 **Chatbot-style Resume Builder** – Create resumes by entering details through guided commands
* 📄 **Automatic Resume Generation** – Converts user input into a structured resume
* ⬇️ **PDF Download** – Users can download their resume in PDF format
* 📊 **ATS Score Checker** – Upload a resume to check ATS compatibility
* 🎨 **User-Friendly Interface** – Simple and clean design for easy usage

## ⚙️ How It Works

1. User opens the Resume Buddy website.
2. The chatbot-style interface asks for details such as:

   * Personal information
   * Education
   * Skills
   * Experience
3. The system processes the data and generates a resume.
4. The user can **download the resume as a PDF**.
5. Users can also **upload an existing resume** to check its **ATS score**.
